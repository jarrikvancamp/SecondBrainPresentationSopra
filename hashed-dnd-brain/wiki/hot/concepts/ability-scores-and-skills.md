Every creature in D&D has six ability scores that measure physical and mental characteristics; each score generates a modifier used in [[playing-the-game|D20 Tests]].

## The Six Abilities

| Ability | Abbreviation | Measures |
|---------|:---:|---------|
| Strength | STR | Physical might |
| Dexterity | DEX | Agility, reflexes, and balance |
| Constitution | CON | Health and stamina |
| Intelligence | INT | Reasoning and memory |
| Wisdom | WIS | Perceptiveness and mental fortitude |
| Charisma | CHA | Confidence, poise, and charm |

## Ability Score Range & Modifiers

Scores normally run 1–20 for player characters (up to 30 for monsters). The modifier = (score − 10) ÷ 2, rounded down.

| Score | Modifier | Score | Modifier |
|:---:|:---:|:---:|:---:|
| 1 | −5 | 16–17 | +3 |
| 2–3 | −4 | 18–19 | +4 |
| 4–5 | −3 | 20–21 | +5 |
| 6–7 | −2 | 22–23 | +6 |
| 8–9 | −1 | 24–25 | +7 |
| 10–11 | +0 | 26–27 | +8 |
| 12–13 | +1 | 28–29 | +9 |
| 14–15 | +2 | 30 | +10 |

## Skills (Mapped to Ability)

Each skill ties to one ability by default. Proficiency in a skill adds your Proficiency Bonus to checks using that skill.

| Skill | Ability | Common Uses |
|-------|---------|-------------|
| Athletics | STR | Climbing, jumping, swimming, grappling |
| Acrobatics | DEX | Balance, tumbling, acrobatic stunts |
| Sleight of Hand | DEX | Pickpocketing, concealing objects |
| Stealth | DEX | Moving quietly, hiding |
| Arcana | INT | Recall lore on spells, magic items, planes |
| History | INT | Recall lore on events, people, cultures |
| Investigation | INT | Deduce how things work, find obscure clues |
| Nature | INT | Recall lore on terrain, plants, animals |
| Religion | INT | Recall lore on gods, rituals, holy symbols |
| Animal Handling | WIS | Calm or direct animals |
| Insight | WIS | Read a person's mood and intentions |
| Medicine | WIS | Diagnose illness, assess cause of death |
| Perception | WIS | Notice things through any sense |
| Survival | WIS | Track, forage, navigate, avoid hazards |
| Deception | CHA | Lie convincingly, wear a disguise |
| Intimidation | CHA | Threaten or awe someone |
| Performance | CHA | Act, sing, dance, tell stories |
| Persuasion | CHA | Honestly convince someone |

> The DM may allow a skill to pair with a different ability when the fiction calls for it (e.g., Strength (Intimidation) for a show of brute force).

## Passive Checks

A passive check is a fixed score used when no roll is called for (e.g., noticing something while not actively searching).

**Passive Score = 10 + all modifiers that normally apply to the check.**

With Advantage add 5; with Disadvantage subtract 5. Most commonly used for Passive Perception.

## Determining Ability Scores

Players use one of three methods at character creation: Standard Array (15/14/13/12/10/8), Point Buy (27 points), or random rolls (4d6 drop lowest, six times). Background then adjusts two or three scores (none above 20).

The D20 Test is the central resolution mechanic of D&D: roll a d20, add modifiers, compare the total to a target number to determine success or failure.

## The Three Kinds of D20 Test

| Type | Purpose | Target Number |
|------|---------|---------------|
| Ability Check | Overcome a challenge using talent or training | Difficulty Class (DC) set by the DM |
| Saving Throw | Resist or evade a hazard or effect | DC set by the effect or spell |
| Attack Roll | Determine if an attack hits a target | Target's Armor Class (AC) |

## How a D20 Test Works

1. Roll 1d20 (always want high).
2. Add the relevant **ability modifier**.
3. Add your **Proficiency Bonus** if you are proficient in the skill, save, or weapon involved.
4. Add any other circumstantial bonuses/penalties.
5. If total ≥ target number → success; otherwise → failure.

## Difficulty Class (DC) Reference

| Task Difficulty | DC |
|---|---|
| Very easy | 5 |
| Easy | 10 |
| Medium | 15 |
| Hard | 20 |
| Very hard | 25 |
| Nearly impossible | 30 |

## Advantage and Disadvantage

- **Advantage:** Roll 2d20, use the **higher** result.
- **Disadvantage:** Roll 2d20, use the **lower** result.
- Multiple sources of Advantage or Disadvantage don't stack — always just two dice.
- If a roll has both Advantage and Disadvantage simultaneously, they cancel out regardless of how many sources.

## Proficiency Bonus

All creatures have a Proficiency Bonus that scales with level or Challenge Rating. Added to D20 Tests only when proficiency applies; never added twice.

| Level/CR | Bonus |
|:---:|:---:|
| Up to 4 | +2 |
| 5–8 | +3 |
| 9–12 | +4 |
| 13–16 | +5 |
| 17–20 | +6 |

**Expertise** doubles the Proficiency Bonus for specific checks; the bonus can only be multiplied once and divided once per roll.

## Attack Rolls Specifically

- **Natural 20:** Automatic hit; Critical Hit (double damage dice).
- **Natural 1:** Automatic miss, regardless of modifiers.
- Melee attacks use Strength; ranged attacks use Dexterity; spell attacks use the caster's spellcasting ability.

## Saving Throws

Named for the ability used (e.g., Constitution save, Wisdom save). Each class grants proficiency in two saving throws. You may voluntarily fail a save if you choose not to resist an effect.

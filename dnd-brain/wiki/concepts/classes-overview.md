# Classes Overview

A character class is the primary definition of what an adventurer can do — it
determines hit points, proficiencies, spellcasting ability (if any), and the
special features that grow with each level.

## The Twelve Classes

The 2024 PHB contains exactly twelve classes. Every class offers four subclasses,
each representing a distinct specialisation within the broader archetype. Subclasses
are chosen at level 3.

## Class Mechanics

| Mechanic | Description |
|---|---|
| **Hit Die** | The die rolled to determine HP gained per level |
| **Primary Ability** | The ability score most important to the class; also the multiclass prerequisite (score ≥ 13) |
| **Saving Throw Proficiencies** | The two saving throws the class trains in from level 1 |
| **Spellcasting** | Whether the class casts spells, and what type (full / half / subclass-only / none) |
| **Subclasses** | Four per class; chosen at level 3; deepen or redirect the class's core fantasy |

## Spellcasting Categories

- **Full casters** (all class levels count toward spell slots): Bard, Cleric, Druid, Sorcerer, Wizard
- **Half casters** (half levels count): Paladin, Ranger
- **Subclass casters** (one-third of levels, specific subclass required): Fighter (Eldritch Knight), Rogue (Arcane Trickster)
- **Pact Magic**: Warlock — uses short-rest-refreshing slots rather than the standard progression
- **Non-casters**: Barbarian, Monk
